# docker-jdk-mvn-fabric

Docker container definition containing:
- Oracle Java 8,
- Maven
- Fabric (Fabric 1.11.1, Paramiko 1.17.1)
- Git 2.7

Used for testing (mvn test) and deploying (fabric) Spring application
